# ShadowOS
A Locked OS for Creators
A closed and locked operating system built solely for creators who are tired of the predefined world and want to create a world of their own. 
If you are tired of old stereotypes or are looking for a way to create, ShadowOs is definitely for you. 
This operating system contains dedicated tools just for creation. From a dedicated game engine to a web browser, everything is within this operating system and the only thing that can stop you is your imagination. 
Install this OS without any problems and enjoy it.
